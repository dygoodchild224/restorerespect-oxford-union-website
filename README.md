# Restore Respect Oxford Union Website

Welcome to the official repository for the [Restore Respect Oxford Union](https://www.restorerespectoxfordunion.co.uk/) website. This website was built using Squarespace to serve as a central platform for outreach, advocacy, and information sharing around the Oxford Union presidential election.

## Project Overview

The website was created to inform and engage members of the Oxford Union before, during, and after the election process. Its primary goals were to:

- Provide clear information about the presidential election process  
- Raise awareness about the controversy surrounding the president-elect‚Äôs comments on the shooting of Charlie Kirk  
- Mobilize members by explaining the reasons behind calls for his removal  
- Guide users on how voting was to be carried out  
- Share updates, news, and opinions via the blog section  

The site served as both an advocacy platform and an informational hub for the membership community.

## Key Features

- Fully responsive design for desktop and mobile  
- Sections include About, Programs, News/Blog, Contact, Gallery, and Resources  
- Blog section for publishing articles, updates, and commentary  
- Information on voting procedures and timelines  
- Integration with social media feeds for broader reach  

## Technology Stack

- **Platform:** Squarespace  
- **CMS:** Squarespace native  
- **Languages Used:** HTML, CSS, JS (through Squarespace custom code blocks)  
- **Hosting:** Squarespace managed hosting  
- **Analytics & SEO:** Google Analytics, Squarespace SEO tools  

## Deployment
