# Restore Respect Oxford Union Website

Welcome to the official repository for the [Restore Respect Oxford Union](https://www.restorerespectoxfordunion.co.uk/) website. This website was built using Squarespace to provide an interactive and informative platform showcasing the initiatives and advocacy of the Restore Respect project.

## Project Overview

The website serves as:
- A hub for advocacy and awareness
- A resource center for women and children affected by conflict-related sexual violence
- A platform for news, events, and updates

**Key Features:**
- Fully responsive design for desktop and mobile
- Interactive sections including About, Programs, News/Blog, Contact, Gallery, and Resources
- Donation system and newsletter subscription
- Integration with social media feeds

## Technology Stack

- **Platform:** Squarespace
- **CMS:** Squarespace native
- **Languages Used:** HTML, CSS, JS (through Squarespace custom code blocks)
- **Hosting:** Squarespace managed hosting
- **Analytics & SEO:** Google Analytics, Squarespace SEO tools

## Deployment

The website is deployed on Squarespace and accessible via [https://www.restorerespectoxfordunion.co.uk/](https://www.restorerespectoxfordunion.co.uk/).
